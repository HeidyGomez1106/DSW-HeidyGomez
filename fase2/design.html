Propósito

El propósito de este sitio web es dar a conocer las estructuras de las etiquetas semánticas para que los motores de búsqueda o cualquier otro mecanismo automático que lea un sitio web sepa con precisión qué partes de sus contenidos pertenecen a cada una de las partes de un sitio.

Etiquetas semánticas

Concepto

Semántica se refiere a todo aquello que tiene que ver con el significado de una palabra u expresión. Pasando al terreno HTML, dentro de él existen etiquetas con significado semántico, etiquetas que por sí mismas brindan un poco de información sobre qué tipo de contenido hay dentro de ellas. Estas ayudan a la implementación de los estándares de la Web Semántica , cuya finalidad es hacer el contenido de internet legible para aplicaciones informáticas. Uno de los usos mas importantes de este tipo de etiquetas es la de marcar la informacion de la pagina para ser indexada por los buscadores web como Google.

Algunas etiquetas semánticas

    aside: Define contendio relacionado con el contenido principal, pero no froma parte relevante para él.
    article: Representa una sección de contenido que puede ser distribuida de forma independiente.
    main: Define el contenido principal de un documento.
    footer: Define el píe de página o sección.
    section: Define una sección en concreto del documento.
    nav: Elemento que contine links de navegación.

CSS Grid

Concepto

Es una técnica de Hojas de Estilo en Cascada, que permite a los desarrolladores de las Webs crear diseños adaptables y complejos con mayor facilidad en todos los navegadores.

Características para posicionar los elementos en la página:

    Ofrece un control total de las dos dimensiones.
    La ubicación de los ítems es libre; no es una tabla.
    Se pueden colocar los ítems donde lo necesitemos, en cualquiera de las celdas que el Grid describe.
